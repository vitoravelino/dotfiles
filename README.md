# Dotfiles & packages

This is an `openSUSE Leap 15.1` based dotfiles & packages.

Install `git` and `zsh`:

```sh
$ sudo zypper in git zsh
```

Switch shell to `zsh`:

```sh
$ chsh -s /bin/zsh
```

Clone the repo:

```sh
$ git clone https://github.com/vitoravelino/dotfiles.git
```

Run `install.sh` script:

```sh
$ ./install.sh
```

Be happy! :)
